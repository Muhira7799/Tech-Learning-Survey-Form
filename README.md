# Tech Learning Survey Form

## Overview
This project is a responsive survey form built with HTML5 and CSS3. It collects information about users' learning preferences, experience level, favorite learning platform, and technologies they use.

## Features
- Responsive and modern user interface
- Personal information fields
- Email and age validation
- Dropdown selection menu
- Radio buttons for experience level
- Checkboxes for technology preferences
- Comment section for additional feedback
- Styled submit button with hover effect

## Technologies Used
- HTML5
- CSS3

## What I Learned
While building this project, I learned how to:
- Create accessible HTML forms.
- Validate user input with HTML5 form attributes.
- Style forms using CSS.
- Build responsive layouts with the CSS box model.
- Use Flexbox-friendly spacing and modern UI design techniques.
- Improve the user experience with hover effects and clean styling.

## How to Run
1. Clone or download the project.
2. Open `index.html` in any modern web browser.
3. Complete the survey and explore the responsive layout.
